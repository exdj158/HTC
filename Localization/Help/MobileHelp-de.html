Web-Dateimanager, Interface für mobile Geräte. Benutzerhandbuch

 Für iOS Benutzer (iPhone, iPad):

  • Sie können die mobile Anwendung (Client für den Web-Dateimanager) für iPhone oder iPad installieren. Die Anwendung ermöglicht es, den Pfad zum Server sowie den Benutzernamen und das Passwort zu speichern, sodass Sie diese nicht jedes Mal neu eingeben müssen. Sie können Die Anwendung aus dem AppStore herunterladen: http://itunes.apple.com/us/app/http-commander/id561413475?mt=8
    Screenshot

     

  • Sie können Dokumente in der mobilen Anwendung Pages bearbeiten. Diese Anwendung ermöglicht es, Ihre Ordner einzubinden und danach schnell und einfach Dokumente bearbeiten und diese auf den Server direkt von Ihrem iPhone/iPad hochladen. Nach der Bearbeitung können Sie die Dokumente zurück auf den Server hochladen.
    • Installieren Sie die Anwendung Apple Pages auf Ihr mobiles Gerät: https://itunes.apple.com/us/app/pages/id361309726?mt=8 .
    • Starten Sie die Anwendung Pages
    • Um das Dokument vom Server zu öffnen: Klicken Sie auf das "+" Symbol in der oberen linken Ecke, wählen Sie danach "Copy from WebDAV" im Menü. Geben Sie im erscheinenden Dialog "WebDAV Sign In" die Serveradresse:  , Ihr Login und Ihr Passwort. Klicken Sie auf den Knopf "Sign In". Danach soll die Liste all Ihrer Ordner angezeigt werden und Sie können das zu bearbeitende Dokument auswählen.
    • Um das Dokument auf den Server zu laden oder Änderungen im Dokument zu speichern: Finden Sie im Bearbeitungsmodus des Dokuments das Zeichen Tools icon im Toolbar und klicken Sie danach darauf, wählen Sie danach den Punkt "Share and print" aus, danach"Copy to WebDAV" (oder "Send copy" und dann WebDAV). Geben Sie im erscheinenden Dialog "WebDAV Sign In" die Serveradresse ein:   und Ihr Login und Passwort. Klicken Sie auf den Knopf "Sign In". Danach soll eine Liste Ihrer Ordner angezeigt werden. Speichern Sie das Dokument mit dem neuen Dateinamen oder benutzen Sie denselben Namen wie beim geöffneten Dokument, um ihn auf dem Server zu aktualisieren.

 For Android users:

  • Sie können die mobile Anwendung (Client für den Web-Dateimanager) für Ihr Android-Gerät installieren. Die Anwendung ermöglicht es, den Pfad zum Server sowie den Benutzernamen und das Passwort zu speichern, sodass Sie diese nicht jedes Mal neu eingeben müssen. Sie können die Anwendung aus dem Android PlayMarket herunterladen: https://play.google.com/store/apps/details?id=com.elementit.httpcommander
    Screenshot

     

  • Ebenfalls können Sie die populäre kostenlose Anwendung ES File Explorer benutzen, um Ihre Dateien und Ordner zu verwalten. Mit Hilfe dieser Anwendung können Sie Ihre Dateien und Ordner in einem für Sie gewohnten Programm verwalten. Sie können allgemeine Operationen an Dateien und Ordnern durchführen und dieser hoch- und herunterladen. Installationsanleitung:
    • Laden Sie die Anwendung ES File Explorer auf Ihrem mobilen Gerät aus dem PlayMarket herunter und installieren Sie diese: https://play.google.com/store/apps/details?id=com.estrongs.android.pop
    • Starten SIe die Anwendung ES File Explorer
    • Klicken Sie auf das Symbol ("View") in der linken oberen Ecke und Wählen Sie FTP in der Gruppe "Netz"
    • Klicken Sie auf den Knopf "New" in der unteren Leiste, wählen Sie im erscheinenden Menü den Punkt "WEBDAV".
    • Füllen Sie alle Felder aus (Geben Sie im Feld "Serveradresse" die Zeile "" ein, füllen Sie die Felder "Login" und "Passwort" aus) und Klicken Sie auf "OK". Das neue Element erscheint in der Liste der Server im FTP-Fenster.
    • Klicken Sie auf das hinzugefügte Element, um sich mit dem Server zu verbinden.
    • Nach der Verbindung müssen Sie die Liste Ihrer Dateien und Ordner sehen. Jetzt können Sie mit der Arbeit mit Ihren Dateien anfangen.
  • Sie können die Dokumente direkt auf dem Server mit Hilfe der Anwendung Kingsoft Office bearbeiten. Diese Anwendung ermöglichtes Ihnen, Ihre Ordner schnell anzuschließen und danach die Dokumente schnell und einfach direkt auf Ihrem Android-Gerät zu bearbeiten. Nach der Bearbeitung werden alle Änderungen automatisch auf dem Server gespeichert.
    • Installieren Sie die Anwendung Kingsoft Office auf Ihr Gerät: https://play.google.com/store/apps/details?id=cn.wps.moffice_eng .
    • Starten Sie die installierte Anwendung
    • Um das Dokument auf dem Server zu öffnen: Klicken Sie auf Open -> Cloud Storages im oberen Menü, wählen Sie in der danach auftauchenden Liste "Add WebDAV / FTP".
      Geben Sie im Dialogfenster die URL zum Server: und den Namen für diese Verbindung (ihrer Wahl). Klicken Sie auf den Knopf "Submit". In der Verbindungsliste sollte die eben gerade von Ihnen erstellte Verbindung erscheinen, klicken Sie darauf und geben Sie Ihren Login und Ihr Passwort ein.
    • Wenn alles richtig gemacht wurde, müssen Sie die Liste all Ihrer Ordner und Dateien sehen. Klicken Sie einfach auf die benötigte Datei, um sie zur Bearbeitung oder zur Ansicht zu öffnen.
    • Um das Dokument auf dem Server zu speichern oder die Änderungen im Dokument beizubehalten: wenn Sie das Dokument vom Server WebDAV geöffnet haben, finden und klicken Sie auf den Knopf Save auf der Symbolleiste. Wenn Sie ein lokales (auf dem Gerät gespeichertes) Dokument geöffnet haben oder wenn Sie das Dokument unter einem anderen Namen speichern wollen, finden und klicken Sie auf den Knopf Save As auf der Symboleiste. Wechseln Sie im erscheinenden Dialog zum Tab 'Cloud storaged' im oberen Teil des Dialogs und wählen Sie aus der Liste die benötigte Verbindung (wenn Sie sie noch erstellt haben, schauen Sie sich den vorherigen Punkt des Öffnens des Dokumentes an) und danach den Ordner auf dem Server.
