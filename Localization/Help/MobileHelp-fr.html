Manageur web-file, l'interface pour les appareils mobiles. Manuel d'usager

 Pour les usagers des appareils sur la base de iOS (iPhone, iPad):

  • Vous pouvez installer une application mobile (client pour le manageur du web-file) pour iPhone ou iPad. L'application permet à maintenir le chemin au serveur et aussi le nom et la parole donc il ne faut pas les entrer chaque fois. Vous pouvez charger une application du App Store http://itunes.apple.com/us/app/http-commander/id561413475?mt=8
    Screenshot

     

  • Vous pouvez corriger des documents dans une application mobile Pages. Cette application permet de connecter vos dossiers et puis corriger des documents vite et facilement et les charger sur le serveur directement de votre iPhone/iPad. Après la correction vous pouvez charger les document sur le serveur.
    • Installez l'application Apple Pages sur votre appareil mobile: https://itunes.apple.com/us/app/pages/id361309726?mt=8 .
    • Lancez l'application Pages
    • Pour ouvrir un document du serveur: Pressez le symbole + dans le coin haut gauche, puis choisissez "Copy from WebDAV" dans le menu. Dans le dialogue apparu "WebDAV Sign In" entrez une adresse du serveur:  , vos nom et parole. Pressez le bouton "Sign In". Après cela on va montrer la liste de vos dossiers et vous pouvez choisir un document nécessaire à corriger.
    • Pour charger un document sur le serveur ou sauver les changements dans le document: Dans le régime de la correction du document trouvez et pressez le symbole Tools icon sur les barres d'outils, puis choisissez le point "Share and print" , puis "Copy to WebDAV" (ou "Send copy", et puis WebDAV). Dans le dialogue apparu "WebDAV Sign In" entrez une adresse du serveur:  et vos nom et parole. Pressez le bouton "Sign In". Après cela vous allez voir la liste de vos dossiers. Sauvez le document sous un nom nouveau du file ou utilisez le même nom du document ouvert pour le renouveler dans le serveur.

 Pour usagers d'Android:

  • Vous pouvez installer une application mobile (client pour le manageur du web-file) à votre appareil sur Android. L'application permet à maintenir le chemin au serveur et aussi le nom et la parole donc il ne faut pas les entrer chaque fois. Vous pouvez charger l'application de Android Play Market https://play.google.com/store/apps/details?id=com.elementit.httpcommander
    Screenshot

     

  • Aussi vous pouvez user une application populaire gratuite ES File Explorer pour admimistrer vos files et dossiers. À l'aide de cette application vous pouvez diriger vos files et dossiers dans un programme habituel. Vous pouvez executer les opérations principales avec des files et dossiers et aussi les charger. L'instruction de l'ajustement:
    • Chargez et installez l'application ES File Explorer sur votre appareil mobile du Play Market https://play.google.com/store/apps/details?id=com.estrongs.android.pop
    • Lancez l'application ES File Explorer
    • Pressez le symbole ("View") dans le coin gauche haut et choisissez FTP dans le groupe "Réseau"
    • Pressez le bouton "New" sur les barres basses , dans le meny apparu chosissez "WEBDAV".
    • Remplissez toutes les lignes (Dans la ligne 'adresse du serveur' entrez "", remplissez les lignes nom et parole) et pressez le bouton "OK". Un élément nouveau va apparaître dans la liste des serveurs de la fenêtre FTP.
    • Pressez un élément ajouté pour connecter au serveur.
    • Après la connexion vous allez voir la liste des dossiers et des files. Maintenant vous pouvez commencer travailler avec vos files.
  • Vous pouvez corriger des documents directement sur le serveur à l'aide d'une application Kingsoft Office. Cette application permet de brancher vos dossiers et puis corriger des documents vite et facilement directement sur votre appareil sur Android. Après la correction tous les changements seront sauvés sur le serveur.
    • Installez l'application Kingsoft Office sur votre appareil: https://play.google.com/store/apps/details?id=cn.wps.moffice_eng .
    • Lancez une application installée
    • Pour ouvrir un document du serveur: Pressez Open -> Cloud Storages dans le haut menu, puis choisissez "Add WebDAV / FTP" dans la liste apparue.
      Dans la fenêtre de dialogue entrez URL au serveur: et un nom à cette connexion (à votre choix). Pressez le bouton "Submit". Dans la liste des connexions on va apparaître le joint créé, pressez le et entrez vos nom et parole.
    • Si tout est fait bien vous allez voir la liste de vos dossiers et files. Simplement pressez le document nécessaire pour l'ouvrir à corriger ou regarder.
    • Pour sauver le document au serveur ou sauver les changements dans le document: Si vous avez ouvert le document au serveur WebDAV, trouvez et pressez Save sur les barres d' outils. Si vous avez ouvert un document local (sauvé sur l'appareil) ou s'il faut sauver un document sous un autre nom, trouvez et pressez le bouton Save As sur les barres d' outils. Dans le dialogue apparu choisissez une feuille 'Cloud storaged' dans une part haute du dialogue et choisissez sur la liste la connexion nécessaire (si vous ne l'avez pas encore créé, voyez le point précédent de l'ouverture du document), et puis le dossier sur le serveur.
