Web file manager, l'interfaccia per i dispositivi mobili. Istruzioni d'uso

 Per gli utenti dei dispositivi sulla base dei iOS (iPhone, iPad):

  • Potete installare l’applicazione mobile (l’utente per il web file manager) per iPhone o iPad. Il mobile app permette di mantenere la via al server ed anche l’username e la password in modo da accedere subito senza la necessita` di scriverli ogni volta all’entrata. Potete caricare il mobile app da App Store http://itunes.apple.com/us/app/http-commander/id561413475?mt=8
    Screenshot

     

  • Potete modificare i documenti nell'applicazione mobile Pages. Questa applicazione permette di connettere alle vostre cartelle e successivamente modificare i documenti velocemente e senza sforzo e caricarli sul server direttamente dal vostro iPhone/iPad. Dopo la revisione potrete caricare i documenti di nuovo sul server.
    • Installare l'applicazione Apple Pages sul vostro dispositivo mobile: https://itunes.apple.com/us/app/pages/id361309726?mt=8 .
    • Avviare l'applicazione Pages
    • Per aprire il documento dal server: Cliccate sul segno + nell'angolo superiore di sinistra successivamente scegliete "Copy from WebDAV" nel menu. Nella finestra del dialogo visualizzata "WebDAV Sign In" inserite l'indirizzo del serever:  , i vostri username e password. Cliccate su "Sign In". Successivamente deve visualizzarsi l'elenco delle vostre cartelle e voi potrete sceglere il documento necessario per la revisione.
    • Per caricare il documento sul server oppure salvare le modifiche nel documento : Nella modalita` della revisione del documento trovate e cliccate sul segno Tools icon sulla barra degli strumenti in seguito scegliete il punto "Share and print" , dopo "Copy to WebDAV" (o "Send copy", e poi WebDAV). Nella finestra del dialogo visualizzata "WebDAV Sign In" inserite l'indirizzo del server:  e i il vostro username e la password. Cliccate su "Sign In". In seguito deve apparire l'elenco delle vostre cartelle. Salvate il documento con il nome del file nuovo oppure usate quello precedente , quello che ha il documento aperto, per rinnovarlo sul server.

 For Android users:

  • Potete installare l’applicazione mobile (l’utente per il web file manager) per il vostro dispositivo basato su Android. Il mobile app permette di mantenere la via al server ed anche l’username e la password in modo da accedere subito senza la necessita` di scriverli ogni volta all’entrata. Potete caricare il mobile app dall’Android Play Market https://play.google.com/store/apps/details?id=com.elementit.httpcommander
    Screenshot

     

  • Potete anche utilizzare l'applicazione gratuita popolare ES File Explorer , per amministrare i vostri file e le cartelle. Con l'aiuto di questa applicazione potete amministrare i vostri file e le cartelle nel programma abituale. Potrete eseguire l'operazioni principali con i file e le cartelle e anche scaricarli e caricarli. Istruzioni per la configurazione:
    • Scaricate e installate l'applicazione ES File Explorer sul vostro dispositivo mobile dal magazzino Play Market https://play.google.com/store/apps/details?id=com.estrongs.android.pop
    • Avviate l'applicazione ES File Explorer
    • Cliccate sul segno ("View") nell'angolo superiore sinistro e scegliete FTP in gruppo "Rete"
    • Cliccate su "New" sulla barra inferiore, nel menu apparso scegliete "WEBDAV".
    • Compilate tutti i punti (Nell'area l'indirizzo del server inserite la riga "", compilate i punti dell'username e della password) e cliccate su "OK". Il nuovo elemento apparira` nell'elenco dei server della finestra FTP.
    • Per connettersi al server cliccate sull'elemento aggiunto.
    • Dopo la connessione dovete vedere l'elenco delle cartelle e dei file. Adesso potete cominciare il lavoro con i vostri file.
  • Potete modificare i documenti direttamente dal server con l’aiuto dell’applicazione Kingsoft Office. Questa applicazione permette di connettere alle vostre cartelle e successivamente modificare i documenti velocemente e senza sforzo direttamente sul vostro dispositivo basato su Android. Dopo le modifiche tutti i cambiamenti automaticamente si salvano di nuovo sul server.
    • Installate l’applicazione Kingsoft Office sul vostro dispositivo: https://play.google.com/store/apps/details?id=cn.wps.moffice_eng .
    • Avviate l’applicazione installata
    • Per aprire il documento dal server: Cliccate su Open -> Cloud Storages nel menu di sopra, dopo selezionate "Add WebDAV / FTP" nell’elenco apparso.
      Nella finestra del dialogo inserite URL per il server: e la denominazione per questa connessione (a vostra scelta). Cliccate su "Submit". Nell’elenco delle connessioni deve apparire la connessione appena fatta, cliccate su di essa ed inserite il vostro username e la password.
    • Se tutto e` fatto in modo corretto, voi dovete vedere l’elenco delle vostre cartelle e dei file. Per aprire il documento necessario per la modificazione oppure per la visualizzazione semplicemente cliccate su di esso.
    • Per salvare il documento sul server oppure salvare le modifiche nel documento: Se avete aperto il documento dal server WebDAV, trovate e cliccate su Save sulla barra degli strumenti. Se avete aperto il documento locale (salvato sul dispositivo) oppure se e` necessario salvare il documento sotto un’altro nome torvate e cliccate Save As sulla barra degli strumenti. Nella finestra del dialogo che appare commutate sul tab 'Cloud storaged' nella parte superiore del dialogo e sceliete dall’elenco la connessione necessaria (se non l’avete ancora creato guardare il punto precedente dell’apertura del documento) e successivamente la cartella sul server.
