La navigation

 Les dossiers  Un ou quelques dossiers de racine sont accessibles. Ils sont visualisés à gauche dans la forme d'un arbre. Utilisez le bouton Dossiers sur les barres d'outils pour montrer ou cacher l'arbre des dossiers.
 La liste  À plus droite au centre — il y a une liste du contenu d'un dossier choisi. Pour passer au dossier mis ou paternel cliquez par la souris sur un nom.
 La sélection du file  Vous pouvez choisir un ou quelques files ou dossiers en cliquant les éléments de la liste et usant les boutons Ctrl et Shift. À l'aide du menu contextuel vous pouvez choisir tous les files ou intervertir un élément courant.
 Menu  On peut appeler le menu contextuel en cliquant le bouton droit de la souris ou en pressant le bouton File sur les barres d'outils.

Les opérations principales aux files

Vous pouvez exécuter toutes les opérations principales aux files:
la création, copie, correction, le déplacement, l'élimination, le changement de nom, prévoit, la compression de données et le déballage.

Le chargement des files

 Le chargement

Dépend des ajustements de l'application vous pouvez user quelques façons du chargement des files:
  • Standardisé — le chargement des files l'un après l'autre à l'aide des possibilités incorporées du navigateur (sur le bouton Browse...). Attention! Les versions dernières des navigateurs (Google Chrome, Mozilla Firefox, Microsoft Internet Explorer) supportent la possibilité de la sélection et du chargement de quelques files en même temps. Si vous utilisez le navigateur Google Chrome version 11 et plus, la sélection et le chargement des dossiers sont accessibles.
  • Drag & Drop — dans le cas d'utilisation de la version dernière de votre navigateur vous pouvez ajouter les files par le traînage et puis les charger parallèlement.
  • Java — vous pouvez choisir et charger quelques files et dossiers en même temps.
  • Flash — permet de choisir et charger quelques files en même temps.
  • Silverlight — la sélection et le chargement des quelques files à même temps par parties (cela permet de charger les files du volume plus de 2 Gb). Cette façon permet aussi d'ajouter les files par le traînage.
  • Google Drive — le chargement des files de votre dépôt nuageux  Google Drive.
  • Dropbox — le chargement de vos files du  Dropbox.
  • Microsoft OneDrive — le chargement de files de votre dépôt  OneDrive.
  • Box — le chargement de vos files du  Box.
  • Le chargement sur URL — vous pouvez indiquer l'adresse URL du file dans l'Internet (y inclus sur FTP) ou aux services d'échange des files tels que  Dropbox,  Picasa,  Flickr,  PhotoBucket,  Facebook.
Une note. Pour charger quelques files vous pouvez les mettre aux archives ZIP, puis les charger et remettre, en cliquant le point  Déballer les archives dans le menu contextuel.

 Le chargement

Choisissez un ou quelques files et/ou dossiers, pressez le bouton  Charger (sur les barres d'outils ou au menu contextuel). En dépendant des ajustements vous pouvez charger les files l'au après l'autre sur la liste ouverte ou utiliser Java Chargeur pour charger quelques files et dossiers.
Autres façons du chargement des files/dossiers sont possibles:
  •  Serrer et charger — une façon pour la création automatique des archives des files choisis, des dossiers avec un chargement suivant des archives ZIP.
  •  Les dépôts nuageux → sauver dans...  Google Drive,  Dropbox,  OneDrive,  Box — vous pouvez charger des files, des dossiers aux dépôts nuageux populaires. L'information plus détaillée est représentée dans une partie Les dèpôts nuageux.

 Les références aux files et dossiers

Cette possibilité permet de charger un file vite sans chargement de l'interface principale ou d'ouvrir un dossier nécessaire. Au menu contextuel choisissez le sous-menu Accès commun (ou trouvez un bouton correspondant sur les barres d'outils) et puis le point  Référence sur un file/dossier. Prêtez attention que pour l'usage d'une référence reçue par autres usagers une authentification et un accès à ce file ou dossier sont nécessaires. Dans le cas contraire on va informer d'une faute.

 L'usage commun des files et des dossiers

Vous pouvez créer les références sur des files et des dossiers accessibles aux autres usagers sans autorisation. Pour cela choisissez un file ou dossier nécessaire, puis au menu contextuel choisissez le sous-menu Accès commun (ou trouvez un bouton correspondant sur les barres d'outils) et puis le point  Créer une référence d'accès commun. Cette possibilité donne un accès anonyme au file/dossier. Tout usager même tel qui n'a pas du compte au Web-manageur peut charger des files et ouvrir des dossiers sur cette référence. Telle référence sur un dossier peut aussi présenter aux usagers une possibilité de charger des files, des dossiers (dans une forme des archives ou l'un par l'autre) dans votre catalogue. En créant d'une référence d'accès commun vous pouvez indiquer les e-mails pour vous informer de l'activité d'usage de cette référence.

 La connexion des dossiers web (les références Web Folders)

Les dossiers web permettent d'ouvrir vos catalogues localement dans un système opérationnel en usant le protocole WebDAV. Dans ce cas vous pouvez corriger vos documents localement sans usage du navigateur et nécessité d'entrer chaque fois dans votre compte. C'est assez d'ouvrir un dossier connectée comme un disque de réseau dans votre système opérationnel (notamment dans Microsoft Windows – dans le guide Windows Explorer). On supporte tous les systèmes opérationnels populaires tels que Microsoft Windows, Apple Mac OS, Ubuntu et autres.
Pour la connexion faites les étapes suivants:
  • Choisissez un dossier nécessaire; dans le menu contextuel passez au sous-menu  encore (ou pressez un bouton correspondant sur les barres d'outils) et puis  Connecter un dossier web.
  • Dans le cas si vous usez le système opérationnel Windows copiez une des instructions proposés de la connexion dans une ligne d'instruction Windows (trois instructions peuvent être accessibles: pour des dossiers courant, paternel et tel de racine). Par exemple: net use "*" "http://example.com/htcomnet/hcwebdav/". Puis pressez OK et s'il le faut entrez vos données du compte. Après cela on va apparaître un dossier connecté dans le guide.
  • Pour les systèmes opérationnels Windows un autre façon de la connexion est possible. Copiez une référence mise en gras pour la connexion. Ou copiez une référence au dossier de racine d'une ligne plus bas:
     Puis dans le guide Windows cliquez par un bouton droit de la souris sur une image Ordinateur (ou Cet ordinateur dans Windows 8, 8.1). Choisissez le point Ajouter un dossier de réseau. Dans le dialogue ouvert pressez le bouton Puis. Choisissez un dossier de réseau accordable et pressez le bouton Puis. Dans une ligne d'adresse mettez une référence copiée et pressez le bouton Puis.
  • Dans le Mac OS: dans le menu du programme Finder choisissez Go → Connect to Server, mettez une référence copiée dans une ligne Server address, pressez le bouton Connect; dans le dialogue ouvert choisissez Registered user et pressez le bouton Сonnect.
  • Dans le Ubuntu: passez à Go → Location.... Mettez une référence copiée dans une ligne d'adresse. Dans le cas si une référence commence: avec des symboles http://, remplaces les par dav://, avec des symboles https:// – davs://.
remplaces les aux dav://, avec des symboles https:// - remplacez les aux davs://. Après la connexion du dossier web vous pouvez l'ouvrir encore une fois sans usage du navigateur (même après le rechargement de l'ordinateur). Prêtez attention: si vous avez changé une parole il faut connecter le dossier de réseau encore une fois (à l'exception quand on utilise le contrôle intégré du Windows).

 Les données extras aux files

Vous pouvez voir et indiquer les propriétés, commentaires, descriptions ou autres données extras des files et des dossiers en cliquant dans le menu contextuel le point  Propriétés.
Note! Les données sont sauvées même si les files ou les dossiers seront copiés, déplacés ou renommés.

La révision et la correction des documents, des images

Le manageur web-file a beaucoup de possibilités pour un travail avec des documents.

Voir les documents

Les documents de texte ou PDF, les files des images vous pouvez voir directement dans le navigateur en choisissant le point  Voir / corriger →  Montrer dans le navigateur dans le menu contextuel ou sur les barres d'outils.
Si dans le catalogue il y a des files des images pour les voir dans une forme du slide-show on peut user la possibilité  Voir / corriger →  Voir des images, ou mettre une forme de la liste des files dans le régime du voit des esquisses (le bouton sur les barres d'outils  Vue →  Esquisses).
Vous pouvez aussi user les services-online  Google Docs,  Microsoft Office Web Viewer,  Box View pour voir les documents des formats différents.
Pour cela choisissez un file nécessaire, cliquez dans le menu (ou sur les barres d'outils)  Voir/ correction un point nécessaire (s'il est permis par un administrateur). Puis une référence sera formée en passant sur laquelle vous pouvez regarder le file.
La liste des formats supportés pour regarder dans le service Google Docs: pdf, doc, docx, ppt, pptx, tif, tiff, xls, xlsx, pages, ai, psd, svg, eps, ps, xps, ttf, zip, rar, dxf, txt, css, html, htm, php, c, cpp, h, hpp, js.
La liste des formats supportés pour regarder dans le service Microsoft Office Web Viewer: doc, docx, docm, dotm, dotx, xls, xlsx, xlsb, xlsm, pptx, ppsx, ppt, pps, pptm, potm, ppam, potx, ppsm (un volume des documents Word et PowerPoint ne peut pas excéder 10MB et des documents Excel  – 5 MB).
La liste des formats supportés pour regarder dans le service Box View: pdf, doc, docx, ppt, pptx, xls, xlsx, txt, py, js, xml, css, md, pl, c, m, json.
Aussi vous pouvez convertir de divers types des fichiers avec l'aide du service  CloudConvert ( View / edit →  Convert with CloudConvert).

Les files des images (jpg, jpeg, png, gif, bmp, psd, pxd) peuvent être corrigés à l'aide de service online   Pixlr:  Voir / corriger →  Corriger dans Pixlr. En dehors de la correction vous pouvez créer dans le Pixlr les images nouvelles en cliquant dans le menu contextuel ou sur les barres d'outils une instruction  Créer →  Image dans Pixlr.

Dans ce cas il ne faut pas charger un file après la correction. Microsoft Office, OpenOffice, LibreOffice vont faire cela automatiquement. Les possibilités et ajustements du Microsoft Office:

  • Par défaut informatique le Microsoft Office ne travaille pas quand le contrôle d'authenticité Basic sans une connexion protégée SSL (il ne permet pas d'ouvrir un document ou il l'ouvre dans le régime de lecture). Si pendant un lancement vous avez reçu telle notification vous pouvez corriger cette situation en introduisant les changement dans le registre du Windows. Pour cela faites le suivant (pour une information détaillée adressez à l'article KB2123563):
    • Pour Windows Vista, 7, 8, 8.1; Server 2008, 2012 chargez et lancez le file BasicAuthVista7Set.reg (pour retourner les ajustements précédents faites BasicAuthVista7Remove.reg).
    • Pour Windows XP и Server 2003 il faut exécuter le file BasicAuthXP2k3Set.reg (pour retourner les ajustements – BasicAuthXP2k3Remove.reg).
    • Rechargez l'ordinateur.
  • Attention! Il est vivement conseillé pour Windows XP, Server 2003, Vista, 7 d'installer le renouvellement Microsoft Web Folders (2.3 MB).
  • Microsoft Office supporte un blocage des files pour prévenir la correction simultanée par quelques usagers. Si le file a été bloqué par un autre usager vous ne pouvez pas le changer mais vous pouvez le regarder et quand le file est accessible – vous allez recevoir une notification correspondante.
  • Par défaut Microsoft Office bloque la correction d'un document. Quand la possibilité du changement d'un document est branchée le file est chargé du serveur encore une fois. Pour enlever l'efficacité vous pouvez brancher une possibilité possibilité de corriger un file tout à coup après une ouverture dans les ajustements: dans le menu Microsoft Office choisissez File → Ajustements → Centre de commande → Ajustements du centre de commande... (bouton) → Régime du prévoit protégé, enlevez un signe du point Brancher le régime du prévoit protégé pour les files de l'Internet.
  • Microsoft Office 2010 ne supporte pas les références plus de 255 symboles, c'est pourquoi essayez d'utiliser les noms courts des files avec des symboles ASCII.

Les possibilités et ajustements du OpenOffice, LibreOffice:

Ces applications sont gratuites et accessibles sur les sites OpenOffice, LibreOffice pour nombreux systèmes opérationnels. НLe moment courant OpenOffice ne supporte pas un blocage des files pendant l'usage simultané. LibreOffice supporte partiellement la possibilité du blocage. Pour lancer ces applications on utilise Java-applet. Si vous ne voulez pas permettre un lancement du cet applet, applet vous pouvez copier une référence du file et l'ouvrir directement dans le dialogue d'ouverture des files OpenOffice, LibreOffice.

Les ajustements recommandés pour OpenOffice, LibreOffice:

  • Si pendant une ouverture des documents vous ne voulez pas d'entrer chaque fois vos nom/parole vous pouvez brancher Lancement vite d'OpenOffice (LibreOffice), ) qui sauve une information de l'usager:
    allez à Service → Paramètres → OpenOffice (LibreOffice) → Mémoire et marquez le point Lancer OpenOffice (LibreOffice) pendant un lancement du système.
  • Pour ouvrir et sauver les documents selon une référence (sans Java-applet) il faut brancher un ajustement suivant: Service → Paramètres → OpenOffice (LibreOffice) → User dialogues OpenOffice (LibreOffice).

 Les dépôts nuageux

Dans le menu contextuel ou sur les barres d'outils il y a en dépendant des ajustements) le menu  Dépôts nuageux, qui permet de réagir (charger les files et dossiers) avec des services nuageux populaires du stockage des données:  Google Drive,  Dropbox,  Microsoft OneDrive,  Box.

 Le chargement, le maintien des documents dans Google Drive (le rédacteur online et usage commun des documents)

Vous pouvez sauver les documents dans votre Google Drive ( Dépôts nuageux →  sauver dans... →  Google Drive) ou les charger ( Dépôts nuageux →  charger de... →  Google Drive). Après le maintien des documents dans votre Google Drive, vous pouvez les éditer directement dans le navigateur (à l'aide du Google) et puis les recharger dans le manageur web-file. Pendant une autorisation dans le Google Drive vos données (nom/parole) ne sont pas passés au serveur du manageur web-file.

 Dropbox — est un gratuit service nuageux du stockage des données, de la copie de réserve, de la synchronisation, de l'échange et usage commun des files.

Pour sauver dans le Dropbox choisissez des files, des dossiers nécessaires et usez une instruction  Dépôts nuageux →  sauver dans... →  Dropbox. Pour le chargement —  Dépôts nuageux →  charger de... →  Dropbox. Pendant l'autorisation vos données de compte sont passés seulement aux serveurs de Dropbox.

 Microsoft OneDrive

Vous pouvez enregistrer dans le service nuageux gratuit du stockage des données, de la rédaction et l'échange des documents  Microsoft OneDrive (si vous avez déjà un compte Microsoft, il ne faut pas enregistrer pour un accès au OneDrive!). À l'aide du manageur web-file vous pouvez sauver vos files, dossiers dans votre OneDrive ( Dépôts nuageux →  sauver dans... →  Microsoft OneDrive) ou les charger ( Dépôts nuageux →  charger de... →  Microsoft OneDrive). ). Aussi sur le site OneDrive vous pouvez éditer vos documents et les changer avec autres usagers.

 Box — est un service nuageux du stockage et de l'usage commun des documents

Pour sauver dans le Box choisissez des files, des dossiers nécessaires et usez une instruction  Dépôts nuageux →  sauver dans... →  Box. Pour le chargement —  Dépôts nuageux →  charger de... →  Box.

 Les hyperliens

Le manageur web-file permet d'ouvrir les hyperliens. Ils sont des petits files avec une extension .url. Puis en pressant sur ce hyperlien une nouvelle fenêtre du navigateur (ou une nouvelle feuille) ouvert avec un contenu placé selon cette référence.
Exemple

Pour créer un file du hyperlien (.url) il faut ouvrir Internet Explorer:

  • Ouvrez une web-page nécessaire dans Internet Explorer.
  • Ouvrez le menu du file (pressez le bouton F10 ou Alt, si on ne peut pas le voir), puis File → Envoyer → Label sur bureau.
  • Chargez un label créé dans le manageur web-file.
Aussi vous pouvez changer le nom du file avec un hyperlien mais ne changez pas une extension du file.
Note. Les files .URL sont standardisés pour Windows les files INI avec leur propre extension. Vous pouvez les créer dans un rédacteur de texte si vous n'avez pas Internet Explorer ou vous ne voulez pas l'user.
