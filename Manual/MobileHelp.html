Web based file and documents manager help for mobile devices

 For iOS users (iPhone, iPad):

  • You can install mobile application of this Web file manager to use it at your iPhone or iPad. It allows to save site URL and credentials so you don't need to type them next time. Download application here http://itunes.apple.com/us/app/http-commander/id561413475?mt=8
    Screenshot

     

  • You can edit documents online using Pages application. You can map your web folders and then very easy and quickly to edit or upload documents using your iPhone/iPad. All changes can be saved back to the server.
    • Intall Apple Pages at your mobile device: https://itunes.apple.com/us/app/pages/id361309726?mt=8 .
    • Start Pages application on your mobile device
    • To open document from server: Click plus symbol in the top left corner, then choose "Copy from WebDAV" in menu. In the "WebDAV Sign In" dialog type the Server Address:  and your credentials. Click "Sign In". After that you should see your folders and you can choose document to edit.
    • To save document to the server or to save changes back to the server: Then you edit document find and click Tools icon on the toolbar, then choose "Share and print" in menu and then "Copy to WebDAV". In the "WebDAV Sign In" dialog type the Server Address:  and your credentials. Click "Sign In". After that you should see your folders. Save document with new filename or to the same filename if it was open before from server.

 For Android users:

  • You can install mobile application of this Web file manager to use it at your Android mobile device. It allows to save site URL and credentials so you don't need to type them next time. Download application here https://play.google.com/store/apps/details?id=com.elementit.httpcommander
    Screenshot

     

  • You can also manage your web files using popular ES File Explorer mobile application. It provides main file operations and whole folders upload/download. To install and map your folders:
    • Download and install ES File Explorer at your mobile device from https://play.google.com/store/apps/details?id=com.estrongs.android.pop
    • Start ES File Explorer
    • Click the top left icon ("View") and set FTP view
    • Click "New" button on toolbar, in popup menu select "WEBDAV".
    • Fill in details of connection (Type address "" to webdav folder, your username and password) and press "OK" button. New item should added to the list of servers in ftp view.
    • Now you can connect to server by clicking appropriate item in servers list.
    • You can modify files online, download or upload.
