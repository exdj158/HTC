How to sync web-folder with local folder



There are several ways to perform synchronization of web folders with local directories.
  1. Map Web Folder as local drive and then use any programm for folders synchronization.
    You can use your favorite synch software. For example, for Windows users Total Commander provide built-in support of folders sync.
  2. Use special sync software with WebDav support.
You can choose one of few below:

Download and install software on your PC.

Below you can see instructions to setup folders sync.

CyberDuck

From the main menu of the program select File->Open Connection.
In the new dialog select type of connection WebDav or WebDav (HTTP/SSL).
Paste FULL WebDav url into Server text field , server and path fields will be filled automatically. Uncheck "Anonymous Login" checkbox and enter your cedentials.
Click "Connect" button to establish connection.
After that you can execute synchronization on any folder. Navigate to needed folder in CyberDuck , right mouse click and select "Synchronize.." from conext menu, select destination local folder and configure other synchronization parameters.
For more info use local help according to the program or Online Help.

WinScp

From the main menu of the program select Session->NewSession.
In the new dialog select New Site on the left pane and proceed to configure it on the right pane. Select WebDav as File protocol, select encryption type. Paste FULL WebDav url into Host name text field, host name and remote directory fields will be filled automatically. Enter your cedentials.You can configure local folder in advanced settings.
Click "Login" button to establish connection.
After that you can execute synchronization on any folder. Navigate to needed folder in WinScp, press "Synchronize" button on main toolbar (Ctrl+S) and configure synchronization parameters.
For more info use local help according to the program or Online Help.

Allway Sync

From the main menu of the program select Job->Add new Job.
In the new tab at the left insert the web folder link (see in top text area at this page).
Select folder type - WebDAV Folder below.
By the Configure button nearby on the right specify other parameters of connection, in particular login and the password for access to the web-folder.
On the right in the field specify your local directory for synchronization.
After that you can execute synchronization, and also to set up other parameters, using the Job submenu in the main menu (in particular automatic synchronization).
For more info use local help according to the program or FAQ.

BestSync

From the main menu select Edit->Add Task.
In the new dialog select local folder as Folder1.
Type web folder link (see in top text area at this page) as Folder2(FTP), specify the login/password.
Further you can set up different parameters of synchronization.
For more info use local help according to the program or Q&A.
Screenshot



GoodSync

From the main menu select Job->New.
In the new window enter a job name, select type (Synchronization).
In the appeared tab in the field at the left (by the Browse button) type web folder link (see in top text area at this page), login and the password.
On the right select a local directory for synchronization.
For more info use FAQ or Manual.
