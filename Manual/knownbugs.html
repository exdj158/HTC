Known bugs and limitations

Here is a list of known problems that have no solution.
  • MS Office for Basic authentication asks for authentication each time then you open file to edit. Unfortunately MS Office don't reads authentication from browser and also don't caches it. We can't fix this problem. Also in some situations it may asks for credentials even then user saves file.
    If you check "Remember my password" checkbox in login dialog it still will display the login dialog at next time, but the credentials will be filled in, so you just click "OK".
    If NTLM authentication is used than MS Office asks for credentials only one time when document is opened at first time.

  • Java uploader and download asks for authentication data when you start upload/download. That is a limitation of Java applet, it does not inherit credentials from web browser, you have to authenticate to Java one more time.
