HTTP Commander for Smartphones and Touchpads

HTTP Commander has an interface for mobile devices (Smartphones). It is 100% touchscreen and a very fast interface supported by more than 90% devices: application has 2 mobile versions: Sencha touch-based for iPhone/iPad, Android-based, Blackberry v.6+, Bada and JQuery mobile version for other.

You can access the mobile version in your browser by "/Default.aspx?Mobile=" URL if you configured HTTP Commander at your server or see at our demo server (https://demo.element-it.com/Examples/demoforms/Default.aspx?Mobile= user: demo, password: demo).

Also, there are special mobile applications available for iPhone/iPad and Android. The mobile application is installed by users from the Apple Store or Android Market. It features are 99% the same as you can see in the browser based mobile version but it allows to saving the site URL and credentials so users don't need to type them next time. Download the iOS application here http://itunes.apple.com/us/app/http-commander/id561413475?mt=8

Helpful info:

  • You can view the mobile version also on a PC (non mobile). In Chrome or Safari browsers (both are WebKit based.) you will see the most advanced mobile interface (Sencha-based). Using FireFox or IE you can see the JQuery mobile based version.
  • To make the URL for mobile version more compact, you can setup HTTP redirection. If you like to have it "http://www.yoursite.com/mobilefiles": Create a folder with the name "mobilefiles" at the root of your server. Click its settings and setup HTTP redirection to "HTCOMNET/Default.aspx?Mobile=" (HTTP Commander application location). Related articles for IIS 6 or IIS 7
  • Please remember that most smartphones don't support Windows Integrated authentication so use Basic authentication.
  • You can also get access to files via mobile WebDav software. You should configure Web Folders mapping (links) in the application and use Basic authentication (not Windows Integrated).
  • iOS users (iPhone/iPad) can use the Apple Pages application to edit documents online. They map their folders using WEBDav in Pages and then open files very quickly. The changes are saved back to the server automatically. For more info, see Mobile users Help
